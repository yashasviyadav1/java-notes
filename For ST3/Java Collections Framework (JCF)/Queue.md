## Queue - JCF

declaration :- 
```java
Queue<TreeNode> q = new LinkedList<>(); // Type1
    OR
Queue<Integer> queue = new PriorityQueue<>();  // Type2
```

methods :- 
```java
q.add(element) 

front = q.remove(element) // fetch and remove front of q

front = q.peek() // only fetch front of q

q.size() // to know total elements in q
```
