declaration :-
```java
// 1d

List<List<Integer>> ans = new ArrayList<>(); // 2 d list where each row is individual level of tree

```

methods:-
```java
li.add(element);

li.add(index, element) // add new element at end of index i

li.set(index, element) // replace index i value with element
```
